# Pokeflip!
***

Pokeflip! is a memo game about pokemon, made with JavaScript.

## Installation
***

Use the package manager.

```bash
npm init
```

```bash
$ git clone https://github.com/ayalamarceloruben/comit-project.git
```

## Usage

```bash
$ npm install
$ npm run dev
$ cd ../comit-project/public/index.html
```
